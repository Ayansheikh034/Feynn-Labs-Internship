# AssessAI – Personalized Learning Recommendation System

## Internship Project | AI & Machine Learning Based Educational Platform

---

## Overview

AssessAI is an AI-driven personalized learning platform developed during my internship to design intelligent learning pathways tailored to individual learners.

The system applies machine learning recommendation techniques to analyze learner profiles, educational background, performance history, and learning preferences to generate customized study recommendations.

The project combines **Artificial Intelligence, Recommendation Systems, and Educational Technology (EdTech)** to improve learning engagement and outcomes.

---

## Problem Statement

Traditional education systems follow a one-size-fits-all approach that often fails to address individual learning differences. Learners possess diverse abilities, learning speeds, and career aspirations, resulting in reduced engagement and inefficient learning outcomes.

AssessAI addresses this challenge by building a data-driven system capable of recommending personalized learning paths and identifying suitable academic or career directions.
